
# FraudLeak AI
FraudLeak AI is an internal audit fraud detection agent for enterprise financial transactions. It automatically scans transactions, detects anomalies, assigns risk scores, and generates audit-ready reports.

## Features
- Detect duplicate payments, ghost vendors, and round figures
- Assign risk scores (1-100)
- Optional ML-based anomaly detection
- Generate CSV reports
- Beginner-friendly Python code
